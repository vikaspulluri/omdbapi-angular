    Overview:

  • This is a simple frontend application built with Angular-7, utilizing the omdbapi for fetching the data.
  • This Application lets you search for movies by part of it's title and showcase the results
  • It can keep track of your previous search results as well, which will be showcased just under the original results
  • By selecting any of the result, complete details of the movie will be displayed
  • User can choose the number of results that should display on the screen by pagination controls
